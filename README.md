# Pandas-Churn-Data-Analysis
This project focuses on analyzing customer churn data to identify the factors that influence customer retention and customer loss. Using Python, Pandas, NumPy, the analysis explores customer demographics, subscription details, service usage, payment behavior, and satisfaction levels to uncover actionable business insights.
The objective of this project is to help businesses understand why customers leave, identify high-risk customer segments, and support data-driven strategies that improve customer retention and reduce churn.

Dataset
Dataset Type: Synthetic Customer Churn Dataset,
Records: 1,000 Customers,
Features: 21 Columns,
Target Variable: Churn No,

The dataset contains customer information such as:

Customer ID

Age

Gender

City

Contract Type

Internet Service

Payment Method

Monthly Charges

Total Charges

Customer Satisfaction Score

Customer Tenure

Churn Status

Churn Reason
